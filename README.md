## Available Commands for `Production`

In the project directory, you can run:


### `yarn`

Installs all the dependencies needed for this project to run.


### `yarn build`

Builds the app for production to the `build` folder.<br>
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.<br>
Your app is ready to be deployed!


Now you are reday to serve the build version. You can use `serve` (https://www.npmjs.com/package/serve) package to do that.

to serve defalut port (5000):
### `serve -s build`

to serve specifying the port:
### `serve -s build -l 5005`
